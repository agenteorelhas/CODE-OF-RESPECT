# Code of Respect - Canal de Apoio e Denúncia

Este projeto acadêmico visa fornecer um ambiente seguro e anônimo para vítimas de assédio moral.

## Funcionalidades
- Chat de acolhimento (Simulação/IA).
- Guia de canais de denúncia oficiais.
- Interface focada em privacidade.

## Segurança
- Sanitização de inputs contra XSS.
- Design responsivo e discreto.

## Tecnologias
- HTML5 / CSS3 (Variáveis nativas)
- Particles.js (Efeitos visuais)
- Segurança: Proteção contra XSS via sanitização de inputs.

## Estrutura
- `index.html`: Núcleo da interface.
- `assets/`: Logos e imagens.
