🛡️ Code of Respect — Plataforma de Conscientização Corporativa

Sistema web voltado à identificação, orientação e denúncia de assédio moral no ambiente de trabalho, com foco em anonimato, ética e acesso a órgãos oficiais.

🚀 Visão Geral

O Code of Respect é uma aplicação web que auxilia usuários a reconhecer situações de assédio e tomar decisões conscientes, oferecendo:

📝 Registro de relatos sigilosos
🧠 Autoavaliação interativa
🏛️ Acesso a canais oficiais de denúncia
🔒 Foco em privacidade e segurança
📌 Sobre o Projeto

A plataforma foi desenvolvida como uma solução digital para promover ambientes corporativos mais seguros, permitindo que o usuário:

Relate situações de forma anônima
Avalie possíveis casos de assédio
Seja direcionado para denúncia oficial

O sistema não realiza denúncias automaticamente, apenas orienta o usuário.

🎨 Interface

Interface moderna com tema escuro e elementos visuais dinâmicos.

✨ Características:
Dark mode profissional
Layout centralizado e intuitivo
Animações visuais (background interativo)
Navegação simples (Início, Denunciar, Auto Avaliação)

🧩 Funcionalidades
📝 Relato Interno
Registro anônimo de ocorrências
Campo para descrição detalhada
Simulação de envio para comitê de ética

🧠 Auto Avaliação
Sistema de perguntas interativas
Identificação de possíveis abusos
Feedback baseado nas respostas
🏛️ Órgãos Oficiais

Redirecionamento para o
👉 Ministério Público do Trabalho
Canal seguro para denúncias formais

🔄 Fluxo de Uso
Usuário acessa o sistema
Escolhe entre relatar ou se autoavaliar
Recebe orientação
Pode acessar órgãos oficiais para denúncia
